# Samsung Galaxy S5 (SM-G900F) ANC9 Firmware Archive

This repository contains the earliest **SM-G900F** firmware build I could locate: **G900FXXU1ANC9**.

The firmware is intended **only for the Samsung Galaxy S5 (SM-G900F)**. Do **not** attempt to flash it on other Galaxy S5 variants.

## Notes

* Device: Samsung Galaxy S5 (SM-G900F)
* Firmware build: G900FXXU1ANC9
* Android version: 4.4.2 KitKat
* Flash with Odin or another compatible Samsung flashing tool.
* I recommend verifying the downloaded file with a SHA-256 checksum before flashing.

If you find additional information about this build or discover an even earlier firmware revision for the SM-G900F, feel free to open an issue or contribute to the archive.
